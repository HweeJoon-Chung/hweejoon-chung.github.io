DirectPlay를 사용하여 구현된 간단한 클라이언트 와 서버의 통신을 테스트하기 위한 애플리케이션 구 현에 커맨트 패턴이 적용되었다. 행위패턴 영역의 커 맨드(Command) 패턴은 상속 기법을 이용하여 오퍼 레이션 수행과 구현객체를 분리하는 방법을 제공하 여, 분산작업이 가능하였다. 다수의 사용자가 데이터베이스에 동시에 접속하 는데 있어서 부하를 감소시키기 위해 커넥션 풀 API 의 구현이 검토되었다. 커넥션 풀 API의 구현은 메모 리를 절약시켜주고 액세스 시간을 감소시켜 준다는 효과가 있었고, 커넥션 풀 API의 구현에 싱글톤이 효율적으로 적용되었다.
